5. Practical fraction and stake units
Let:

f_kelly = f* clamped to [0, f_max].

fraction = fractional Kelly multiplier (e.g., 0.25, 0.5, 1.0).

effective_fraction = f_kelly * fraction.

Then:

Typical recommendations:

Full Kelly (fraction=1.0) is mathematically optimal for log utility but can give large drawdowns.

Half or quarter Kelly (fraction=0.5 or 0.25) is common in practice.

6. Confidence tiers
OmegaSportsAgent already uses tiers (e.g., A, B, C) based on:

number of simulations,

calibration confidence,

edge magnitude.

An LLM can apply different fractional Kelly per tier:

Tier A: fraction = 0.5 (half Kelly),

Tier B: fraction = 0.25,

Tier C: fraction = 0.1 or 0.0 (pass except exceptional edges).

Mapping should be configurable per sport and risk appetite.

7. Portfolio and drawdown considerations
If multiple bets are placed in a slate:

Avoid simply summing Kelly stakes as if bets are independent.

cap per-edge: never exceed f_max (e.g., 2% per bet) even if Kelly says more.
